# Rikkipedia

A custom one-page resume

Built using the open-source [json-resume](https://jsonresume.org/) format and [resumed CLI](https://github.com/rbardini/resumed).

Custom theme built in Vue and rendered with Vite

## Installation
```zsh
npm install -g resumed
npm i
```

## Run in dev mode with hot module reloading
```zsh
npm run dev
```
Visit http://localhost:5173/

## Export as single-page HTML
```zsh
npm run export
```
